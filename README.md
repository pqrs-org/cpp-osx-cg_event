# cpp-osx-cg_event

Utilities of `CGEvent`.

## Requirements

cpp-osx-cg_event depends the following classes.

- [pqrs::cf::cf_ptr](https://github.com/pqrs-org/cpp-cf-cf_ptr).
- [pqrs::hid](https://github.com/pqrs-org/cpp-hid).
- [pqrs::osx::iokit_types](https://github.com/pqrs-org/cpp-osx-iokit_types).
- [ethernal](https://github.com/mapbox/eternal).
- [type_safe](https://github.com/foonathan/type_safe).

## Install

Copy `include/pqrs` and `vendor/vendor/include` directories into your include directory.
